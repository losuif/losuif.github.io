## <span style="color:#FF8989">**GROUP BY 절 사용**</span>
<hr>

<br>

```sql
SELECT 
    B.DNAME,
    A.JOB,
    COUNT(*) AS EMP_CNT,
    SUM(A.SAL) AS SAL_SUM
FROM EMP A,
    DEPT B
WHERE B.DEPTNO = A.DEPTNO
GROUP BY B.DNAME, A.JOB;
```

## <span style="color:#FF8989">**GROUP BY 절 + ORDER BY 절**</span>
<hr>

<br>

```sql
SELECT 
    B.DNAME,
    A.JOB,
    COUNT(*) AS EMP_CNT,
    SUM(A.SAL) AS SAL_SUM
FROM EMP A,
    DEPT B
WHERE B.DEPTNO = A.DEPTNO
GROUP BY B.DNAME, A.JOB
ORDER BY B.DNAME, A.JOB;
```
