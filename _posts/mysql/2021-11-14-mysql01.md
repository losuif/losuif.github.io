## <span style="color:#0066ff">MySQL 쿼리(Query)문</span>
<hr>

- 데이터베이스 생성

    - CREATE DATABASE database_name;

- 데이터베이스 삭제

    - DROP DATABASE db_name;

    - DROP DATABASE IF EXISTS db_name;

- 데이터베이스 사용

    - USE db_name;

- 데이터베이스 리스트 조회

    - SHOW DATABASES;

- 현재 사용중인 DB

    - SELECT DATABASE();

    - 현재 작업중인 DB가 없다면 null 반환
