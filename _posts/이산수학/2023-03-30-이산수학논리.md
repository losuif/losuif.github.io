# **<span style="color:#DF7857">🍋 명제 (proposition)</span>**

<hr>

- 참과 거짓을 구별할 수 있는 문장이나 수학적 식

- 명제의 내용이 타당한 경우 : 명제가 참(True, T) / 타당하지 않은 경우 : 명제가 거짓(False, F)

- 참과 거짓 = 명제의 진리값(truth value)

- 참과 거짓을 구별할 수 없는 문장은 명제가 아님
    
<br>
<br>

# **<span style="color:#DF7857">🍋 논리연산 (logical operation)</span>**

<hr>

- 참, 거짓을 구별할 수 있는 명제를 대상으로 하는 연산

- 논리연산자(logical operator) : 각각의 논리연산을 기호로 나타낸 것

<br>
<br>

### 🌱 합성명제(compound proposition) 

- 하나 이상의 명제와 논리연산자들이 결합되어 만들어진 명제

- 합성명제의 진리값은 합성명제에 참여하는 단순명제들의 참(T)과 거짓(F) 여부에 따라 결정됨

- 참여하는 단순명제가 많을 경우 진리표를 사용해 단계적으로 합성명제의 진리값을 구함

- 각각의 단순명제들이 가질 수 있는 모든 진리값을 차례대로 나열, 단순명제에서 주어진 합성명제로 단계적으로 확장해 가면서 진리값을 계산하여 진리표 작성

- 하나의 단순명제는 참 또는 거짓 두 가지 값을 가짐 -> n개의 단순명제로 구성된 합성명제는 2ⁿ개의 경우를 가짐
    
<br>
<br>

### 🌱 기본 논리연산(logical operation)

- 연산자의 우선순위 : ① ~ &nbsp;&nbsp; ② ∧, ∨ &nbsp;&nbsp; ③ →, ↔ &nbsp;&nbsp; ④ ≡

- #### <span style="color:#DF7857">논리합 ( or, ∨ )</span>

    - 연결된 명제들 중 하나라도 참(T)인 경우 참(T), 모두 거짓(F)인 경우 거짓(F)

    - 논리합 _p ∨ q_ 의 진리표

        |_p_|_q_|_p ∨ q_|
        |:---:|:---:|:---:|
        |T|T|T|
        |T|F|T|
        |F|T|T|
        |F|F|F|

- #### <span style="color:#DF7857">논리곱 ( and, ∧ )</span>

    - 연결된 명제들이 모두 참(T)인 경우 참(T), 하나라도 거짓(F)이 있을 경우 거짓(F)

    - 논리곱 _p ∧ q_ 의 진리표

        |_p_|_q_|_p ∧ q_|
        |:---:|:---:|:---:|
        |T|T|T|
        |T|F|F|
        |F|T|F|
        |F|F|F|

- #### <span style="color:#DF7857">부정 ( not, ~ )</span>

    - 따라오는 명제가 참(T)인 경우 거짓(F), 거짓(F)인 경우 참(T)

    - 부정 _~ p_ 의 진리표

        |_p_|_~ p_|
        |:---:|:---:|
        |T|F|
        |F|T|

- #### <span style="color:#DF7857">배타적 논리합 ( xor, ⊕ )</span>

    - 연결된 두 개의 명제 중 1개만 참(T)이면 참(T), 나머지 경우는 거짓(F)

    - 명제 p나 q가 서로 다른 진리값을 가지면 참(T), 동일한 진리값을 가지면 거짓(F) => _p⊕q = (p∧q)∨(~p∧q)_

    - 배타적 논리합 _p⊕q_ 의 진리표

        |_p &nbsp;&nbsp; q_|_p∧ ~q &nbsp;&nbsp; ~p ∧q_|_(p∧ ~q) &nbsp;&nbsp; (~p ∧q)_|_p⊕q_|
        |:---:|:---:|:---:|:---:|
        |T &nbsp;&nbsp; T|F &nbsp;&nbsp; F|F|F|
        |T &nbsp;&nbsp; F|T &nbsp;&nbsp; F|T|T|
        |F &nbsp;&nbsp; T|F &nbsp;&nbsp; T|T|T|
        |F &nbsp;&nbsp; F|F &nbsp;&nbsp; F|F|F|
    
<br>
<br>

### 🌱 조건명제와 쌍조건명제

- #### <span style="color:#DF7857">조건명제(conditional proposition, →)</span>

    - 어떤 사실의 인과관계를 나타내는 명제

    - 명제 p와 q가 있을 때, 명제 p가 조건의 역할을 수행하고, 명제 q가 결론의 역할을 수행하는 경우를 말하며, _p → q_ 로 표시

    - 명제 p가 참이고 q가 거짓인 경우만 거짓, 나머지 경우는 모두 참

    - 조건명제 _p → q_ 의 진리표

        |_p_|_q_|_p → q_|
        |:---:|:---:|:---:|
        |T|T|T|
        |T|F|F|
        |F|T|T|
        |F|F|T|

    - **충분조건**

        - _if p, then q_

        - p가 일어나면 항상 q가 일어나는 것을 보장한다, p는 q의 충분조건

        - _p → q_ 에서 p는 q의 충분조건

    - **필요조건**

        - _if not p, then not q_

        - q가 일어나는 데 p가 필요하다, p는 q의 필요조건

        - _p → q_ 에서 q는 p의 필요조건

- #### <span style="color:#DF7857">쌍조건명제(biconditional proposition, ↔)</span>

    - 두 개의 조건명제가 서로 결합되어 만들어짐

    - 명제 p와 q가 있을 때, p가 조건일 때 q가 결론이 되고, 동시에 q가 조건일 때 p가 결론이 되는 경우를 말하며, _p ↔ q_ 로 표시

    - 쌍조건명제 _p ↔ q_ 는 _(p → q) ∧ (q → p)_ 로 분해 가능

    - p와 q가 동시에 동일한 진리값을 가질 때 참

    - 쌍조건명제 _p ↔ q_ 의 진리표

        |_p_|_q_|_p ↔ q_|
        |:---:|:---:|:---:|
        |T|T|T|
        |T|F|F|
        |F|T|F|
        |F|F|T|

    
<br>
<br>

### 🌱 동치

- #### <span style="color:#DF7857">논리적 동치(logical equivalence, ≡)</span>

    - 두 명제 p와 q가 논리적으로 동등한 경우를 말하며, _p ≡ q_ 로 표시

    - p와 q가 항상 동일한 진리값을 가지면 논리적 동치

    - 두 명제의 진리값은 서로 동일하므로 두 명제를 서로 바꾸어도 무방

    - 쌍조건명제와 논리적동치는 동일한 대상에 대한 다른 표현이기 때문에, 명제들의 관계를 조건-결론 관점에서 이야기하고 싶다면 쌍조건명제를, 논리적 동등성 관점에서 이야기하고 싶다면 논리적 동치를 사용

    - _p ↔ q_, _p = q_, _p ≡ q_, _p ⇔ q_

- #### <span style="color:#DF7857">역, 이, 대우</span>

    - _p → q_ 에 대해서 _q → p_ 를 역(converse), _~ p → ~ q_ 를 이(inverse), _~ q → ~ p_ 를 대우(contrapositive)라고 함

    - 조건명제 _p → q_ 의 역, 이, 대우에 대한 진리표

        |_p_|_q_|_~ p_|_~ q_|_p → q_|_q → p_|_~ p → ~ q_|_~ q → ~ p_|
        |:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
        |T|T|F|F|T|T|T|T|
        |T|F|F|T|F|T|T|F|
        |F|T|T|F|T|F|F|T|
        |F|F|T|T|T|T|T|T|

- #### <span style="color:#DF7857">항진명제, 모순명제</span>

    - 항진명제(tautology) : 항상 참인 명제

    - 모순명제(contradiction) : 항상 거짓인 명제
