## <span style="color:#5cff5c">JSON</span>
<hr>

- JSON : JavaScript Object Notation

- 네트워크를 통해 데이터를 주고 받는 데이터 형식

- key(name) - value 한 쌍

    ```javascript
    {
        "key : name"
    }
    ```

- string / number / boolean / objecet : { } / array : [ ]

- ex)

    ```javascript
    {
        //string
        "phone_number" : "010-1234-5678",

        //number
        "age" : 10,

        //boolean
        "isAgree" : false,

        //object
        "account" : {
            "email" : "1234@gmail.com",
            "password" : "abcd1234"
        }
    }


    //user 조회 하는 경우
    //array
    {
        "user_list" : [
            {
                "account" : "abcd",
                "password: "1234" 
            },
            {
                "account" : "aabb",
                "password: "1212" 
            },

        ]
    }
    ```
